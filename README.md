# 📖vue2-example-demo

该仓库包含了vue2以及与其相关生态的使用技巧。包括但不限于：element-ui的使用技巧、各类JS插件的封装和使用等等......

## 🔧安装
```
npm install
```

## 🚀启动
```
npm run serve
```

## 🔨修复
```
npm run lint
```
